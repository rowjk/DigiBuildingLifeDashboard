# 統一數位大樓生活資訊平台 (Life Dashboard)

本專案是一個為企業內部設計的即時生活資訊儀表板（Dashboard），整合了天氣預報、YouBike 2.0 即時站點、公車到站資訊、公告系統、周邊美食地圖與後台 CRUD 管理系統。

## 技術與架構
* **網頁前端與互動**：Streamlit (純 Python 響應式網頁框架)
* **資料儲存**：SQLite (本機內嵌型資料庫 `dashboard.db`)
* **資料庫 ORM**：SQLAlchemy
* **資料介接與防禦快取**：
  * 使用 Streamlit 全域快取裝飾器限制第三方 API 連線頻率，防止來源 IP 被限制或鎖定。
  * 容錯降級機制：當 API 斷網或資料缺失時，自動顯示最後一次成功快取的暫存資料或 Mock 資料，並在畫面頂端顯示警告。

## 安裝與執行步驟

### 1. 系統需求
* Python 3.10 或以上版本。

### 2. 下載專案與安裝套件
請於終端機（Command Prompt 或 PowerShell）中執行以下指令安裝所需依賴套件：
```bash
pip install -r requirements.txt
```

### 3. 配置環境變數 (`.env`)
本專案嚴格限制將 API 金鑰寫死於程式碼中。請在專案根目錄下建立 `.env` 檔案（已為您自動產生），並配置中央氣象署授權碼：
```env
CWA_API_KEY=您的授權碼
```
*(註：請在專案根目錄下的 .env 檔案中寫入您的個人授權碼，格式如：CWA-XXXXXXXXXXXXXXXX)*

### 4. 初始化資料庫與預建餐廳資料
執行 `database.py` 以建立 `dashboard.db` 資料庫，並預建 **20 筆內湖石潭路周邊精選餐廳**、初始公告與預設管理員帳密：
```bash
python database.py
```
* **預設管理員帳號**：`admin`
* **預設管理員密碼**：`admin` (雜湊 SHA256 加密儲存)

### 5. 啟動 Dashboard 網站
使用 Streamlit 啟動網頁服務：
```bash
streamlit run app.py
```
啟動後，瀏覽器將自動開啟並導向本機網址（通常為 `http://localhost:8501`）。

## 功能展示說明

### 📌 即時生活資訊
1. **今日天氣**：定位台北市內湖區，展示即時溫度、體感溫度、降雨機率與天氣現象。
2. **緊急與重要公告**：北市府公告最新消息與本地公告整合。包含「停班」、「停課」、「颱風」、「地震」等防災關鍵字時，自動高亮為「緊急公告」並於網頁最頂端以紅色 Banner (st.error) 警示。
3. **YouBike 2.0 站點**：即時顯示統一數位大樓、石潭路等周邊站點的可借車數與空位數。
4. **公車即時到站**：即時列出石潭成功路口、新湖一路口、南京金莊路口、石潭金豐街口等站點的行經公車到站剩餘時間。
5. **午餐推薦 Top 10**：提供分類篩選與排序切換（評分高低或距離遠近），自資料庫實時查詢展示前 10 名最合適的周邊餐廳。
6. **生活地圖**：內嵌 Google Maps 互動式地圖，置中於統一數位大樓 (石潭路155號)。

### 📌 後台管理系統 (CRUD)
1. 展開左側側邊欄，勾選「**切換至後台管理**」。
2. 輸入管理員帳號：`admin`、密碼：`admin` 登入。
3. 登入成功後，即可使用視覺化可編輯表格對 SQLite 資料庫中的 **公告資料表** 與 **餐廳資料表** 進行新增、修改與刪除 (CRUD) 操作，儲存後即時生效於主頁。

## 🚀 未來優化與擴充計畫

為了便於日後持續優化此平台，我們規劃了以下擴充方向：

1. **API 串接自動調適與彈性**：
   * 擴展公車與 YouBike API：支援動態輸入起訖點或自動定位，過濾出該使用者最鄰近的站點。
   * 天氣警報主動推送：當偵測到北市府有停班停課或豪大雨特報時，結合 Line Notify、Slack 或 Teams Webhook，即時推送通知至公司內部群組。
2. **後台管理與安全性增強**：
   * 多管理員帳號：提供權限分級，並支援密碼重設與變更功能。
   * 新增審核機制：一般員工可於前台提交推薦餐廳，需經由管理員在後台審核通過後，方可上線。
3. **美食推薦系統優化**：
   * 結合外部 API：串接 Google Maps Platform (Places API)，取得餐廳最新評論與營業時間。
   * 隨機午餐按鈕：「今天吃什麼？」隨機篩選功能，並可搭配轉盤特效，提升互動趣味性。
4. **前端介面設計升級**：
   * 使用 Streamlit 客製化元件（如 `streamlit-shadcn-ui` 或 `streamlit-antd-components`）提升 UI 質感。
   * 實作更精美的主題切換（深色模式、冷暖色調主題）。
